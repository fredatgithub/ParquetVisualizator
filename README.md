## Parquet Visualizator

Une application de visualisation pour fichiers Parquet écrite en C# avec WPF.

## Description

Parquet Visualizator est une application de bureau permettant d'ouvrir et visualiser le contenu des fichiers Parquet de manière intuitive. L'application offre une interface graphique conviviale pour explorer les données structurées dans les fichiers Parquet.

## Fonctionnalités

- Ouverture et lecture des fichiers Parquet
- Affichage des données dans une grille
- Visualisation des métadonnées des colonnes
- Interface utilisateur simple et intuitive

## Prérequis

- .NET Framework 4.8
- Windows (application WPF)

## Installation

1. Clonez le dépôt ou téléchargez les sources
2. Restaurez les dépendances NuGet
3. Compilez la solution ParquetVisualizator.sln
4. Exécutez l'application

## Dépendances

- Parquet.Net 5.4.0 - Bibliothèque pour lire les fichiers Parquet
- System.Text.Json - Sérialisation JSON
- IronCompress - Compression des données

## Utilisation

1. Lancez l'application Parquet Visualizator
2. Ouvrez un fichier Parquet via le menu principal
3. Les données s'affichent dans la grille principale
4. Explorez les colonnes et les métadonnées
